# T1096

**ID:** T1096

**Name:** Run programs identification (OS)

**Description:**

The process of examining operating system files to extract artifacts that may indicate a program has been run.

**Synonyms:**



**Details:**

While this technique may result in executable or script paths and timestamps, to evaluate a hypothesis that a program was run requires use of T1155 to evaluate if this occurred, and optionally T1117 if the time of access is relevant.
