# T1156

**ID:** T1156

**Name:** Functional-based event reconstruction

**Description:**

Reconstructing the configuration or capabilities of a system at the time of an event

**Synonyms:**



**Details:**

Considering how a system works and what conditions were necessary for an event to be possible, i.e. supporting or refuting if a hypothesis is feasible.

This might involve questions such as:
- is a program or system capable of performing some action
- how was the program or system configured (to determine if an operation was possible)
