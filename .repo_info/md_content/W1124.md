# W1124

**Name:** Failure to compute hash correctly: this could result in a message indicating corrupt evidence, thus stopping or delaying further investigation

**Weakness classes:** INAC-AS

**Details:** None

**Mitigations:**

- M1075: Testing programme to validate hashes of data in images is calculated correctly 
- M1085: Use of multiple tools to verify disk image hash 


**References:**

- Hargreaves, C., Nelson, A. and Casey, E., 2024. An abstract model for digital forensic analysis tools-A foundation for systematic error mitigation analysis. Forensic Science International: Digital Investigation, 48, p.301679.
