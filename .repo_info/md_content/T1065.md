# T1065

**ID:** T1065

**Name:** Content indexer examination (OS)

**Description:**

Examine information from operating system artifacts relating to content indexing and search capabilities e.g. Windows Desktop Search

**Synonyms:**



**Details:**

For example the Windows Search database '...contains vast amount of values about various of artifacts found on a Windows system, e.g. files and directories, emails, appointments, attachments, images, audio and video' (Metz 2011)

**Subtechniques:**



**Examples:**

- Windows Search
- Spotlight
- Zeitgeist
