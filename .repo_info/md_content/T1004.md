# T1004

**ID:** T1004

**Name:** Selective data acquisition

**Description:**

Previewing a data source and selecting a subset of the data for collection into a container such as a tar/zip file or a forensic image format.

**Synonyms:**

Logical imaging, 

**Details:**



**Subtechniques:**



**Examples:**

- FTK Imager - allows an AD1 logical container to be created of a selected subset of files within a previewed data source.


**CASE Output Classes:**



**Potential weaknesses and mitigations:**

- W1017: Files or data that is relevant to the investigation is missed _(INCOMP)_
- W1018: Data is collected that is not relevant to the investigation _(INAC-AS)_
- W1019: Data is collected and assigned to incorrect device _(INAC-AS)_
- W1020: Content of data collected is incorrect _(INAC-ALT)_
- W1021: Metadata of data collected is missing _(INCOMP)_
- W1022: Metadata of data collected is incorrect _(INAC-ALT)_
- W1023: Collection process changes content of data on target _(INAC-ALT, INAC-COR)_
    - M1007: Use hardware write blocker ([T1012](T1012.md))
    - M1008: Use software write blocker ([T1013](T1013.md))
- W1024: Collection process changes metadata of data on target _(INAC-ALT, INAC-COR)_
