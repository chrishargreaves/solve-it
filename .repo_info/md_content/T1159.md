# T1159

**ID:** T1159

**Name:** Extract mobile data via deployed agent

**Description:**

Deploy software to a mobile device that can carry out some function, either collect data with available permissions or attempt further exploitation e.g. sandbox escape.

**Synonyms:**



**Details:**

While Android agents can be sideloaded with adb, deploying agents on iOS may require a developer account. Output may be equivalent of a file system acquisition if an exploit is successful. Otherwise data such as contacts would be extracted.

**Subtechniques:**



**Examples:**

- Elcomsoft
- Belkasoft

**Potential weaknesses and potential mitigations:**

- [W1276](W1276.md): Relevant data is overwritten by deploying an agent to a mobile device _(INCOMP)_
    - M1014: Test and document changes made by the tools used
