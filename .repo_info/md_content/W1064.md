# W1064

**Name:** Making incorrect assumptions about sector size e.g. 512 rather than 4096

**Weakness classes:** INCOMP

**Details:** None

**Potential mitigations:**

- M1045: Explicitly detect sector size in use 
- M1046: Check pointer offsets for both 512 and 4096 sectors sizes
