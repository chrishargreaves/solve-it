# T1060

**ID:** T1060

**Name:** Enumerate allocated files and folders

**Objective(s):** Access partitions, volumes and file systems data

**Description:**

This involves parsing the live file system data structures and reconstructing the file system of the target media or device.

**Synonyms:**

file system parsing,

**Subtechniques:**
