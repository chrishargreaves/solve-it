# T1092

**ID:** T1092

**Name:** Produce tag-based automated report

**Description:**

Generate a report in PDF, HTML or other format that compiles tagged (or bookmarked) items from a case into a package that can be reviewed.

**Synonyms:**

Produce bookmark-based automated report,

**Potential weaknesses and potential mitigations:**

- [W1152](W1152.md): Generated report does not include all bookmarks _(INCOMP)_
    - M1104: Testing of report generation functionality that all intended bookmarks are included 
    - M1105: Manual verification that all intended bookmarks are included in the generated report. 
- [W1153](W1153.md): Report generated from bookmarks does not render all content (e.g. previews, string extracts) correctly _(INCOMP, INAC-ALT)_
    - M1106: Testing that generated reports correctly represent/render all expected file types 
    - M1107: Manual verification that all bookmarked items in the report are correctly rendered/represented. 
- [W1154](W1154.md): Report generated from bookmarks stored in manner that does not prevent tampering or corruption _(INAC-COR)_
    - M1108: Preserve hash in secure location of the generated report so intentional or unintentional modifications can be detected.
