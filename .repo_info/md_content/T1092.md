# T1092

**ID:** T1092

**Name:** Produce tag-based automated report

**Objective(s):** Produce documentation

**Description:**

Generate a report in PDF, HTML or other format that compiles tagged (or bookmarked) items from a case into a package that can be reviewed.

**Synonyms:**

Produce bookmark-based automated report,

**Potential weaknesses and potential mitigations:**

- [W1152: Generated report does not include all bookmarks](W1152.md) _(INCOMP)_
    - [M1104: Testing of report generation functionality that all intended bookmarks are included](M1104.md)
    - [M1105: Manual verification that all intended bookmarks are included in the generated report.](M1105.md)
- [W1153: Report generated from bookmarks does not render all content (e.g. previews, string extracts) correctly](W1153.md) _(INCOMP, INAC-ALT)_
    - [M1106: Testing that generated reports correctly represent/render all expected file types](M1106.md)
    - [M1107: Manual verification that all bookmarked items in the report are correctly rendered/represented.](M1107.md)
- [W1154: Report generated from bookmarks stored in manner that does not prevent tampering or corruption](W1154.md) _(INAC-COR)_
