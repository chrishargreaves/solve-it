# W1246

**Name:** Attribution of incorrect details (e.g. page title) to a browser web visit due to join queries used in underlying database

**Weakness classes:** INAC-EX, INAC-AS, MISINT

**Details:** None

**Potential mitigations:**

- M1194: Consider carefully page titles and other details associated with older web history visits when details are recovered using SQL join queries. 
- M1195: Also examine browser cache to confirm details about an older web visit when details are recovered using SQL join queries. ([T1138](T1138.md))
- M1193: Obtain older versions of browser history files from backups or from file system versioning features
