# W1036

**Name:** Overwriting relevant data with changes caused by running the live tools

**Weakness classes:** INCOMP, INAC-ALT

**Details:** None

**Mitigations:**

- M1014: Test and document changes made by the tools used 
- M1015: Use tools only if changes made are to locations where relevant evidence is unlikely to be located. 


**References:**

- Hargreaves, C.J., 2009. Assessing the reliability of digital evidence from live investigations involving encryption. Chapter 5
