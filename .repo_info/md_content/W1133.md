# W1133

**Name:** Tool fails to express uncertainty in the classification of file recovery results, e.g., where content may be partly overwritten.

**Weakness classes:** MISINT

**Details:** None

**Mitigations:**

- M1084: Test that the tool used expresses uncertainty in the classification of file recovery results
