# T1127

**ID:** T1127

**Name:** Keyword search (live) (logical)

**Description:**

Searching for keywords over the raw data, without an index of text content, but on a file by file basis

**Synonyms:**

live search, 

**Details:**

This approach can include decoding the individual file formats and content. It can also overcome limitations associated with compressed data or data split due to file fragmentation.

**Potential weaknesses and mitigations:**

- W1195: Keyword results from a tool are associated with a file, but the result was within slack space and therefore it may not have been part of that file _(INAC-AS, MISINT)_
    - M1151: Check keyword results are part of allocated file, or if not perform detailed analysis to determine if the association between keyword results and file is valid
